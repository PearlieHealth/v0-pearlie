(B) `trackTikTokEvent(event: string, params?: Record<string, unknown>)` — checks consent + `window.ttq` existence, then calls `window.ttq.track(event, params)`. (C) A `declare global` block extending `Window` with `ttq` typed with methods: `load`, `page`, `track`, `identify`. Both functions must return early if `typeof window === "undefined"` (SSR safety).
